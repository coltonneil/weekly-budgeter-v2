DescribeBudget

Gets the data on a budget, and returns a paginated list of Expenses.

path Parameters
budgetId
required
string

Represents the ID of a budget.

query Parameters
startDate
string <date-time> 
Starting timestamp of range to grab expenses within. If provided 'endDate' MUST also be provided.
If not provided, will query all available expenses in budget.
endDate
string <date-time> 
End timestamp of range to grab expenses within. If provided 'startDate' MUST also be provided.
If not provided, will query all available expenses in budget.
maxResults
number [ 1 .. 100 ]

A the max number of results to return in a query.

nextToken
